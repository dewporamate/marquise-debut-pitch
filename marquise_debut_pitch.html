02 — Market Proof & Inspiration

แฟนคลับคาดหวังอะไร

จากการวิเคราะห์ Live References ของ KATSEYE & YUPP! Trainee Debut

REFERENCE 01

KATSEYE — 1st Anniversary Live VOD

HYBE × GEFFEN

Live ฉลองครบรอบ 1 ปีของ KATSEYE บน YouTube — รูปแบบ Sit-Down Interview ผสม Interactive Games เป็นต้นแบบของ "ความเชื่อมต่อที่แท้จริง" กับแฟนคลับ

KEY LEARNINGS
  • Real-Time Comment Interaction
    แฟนคลับรู้สึกเป็นส่วนหนึ่งของงานผ่านการอ่าน Comment ออกอากาศแบบ Real-Time
  • Authentic Unscripted Moments
    ช่วงที่ไม่ได้เตรียมสคริปต์ (Trivia, Games, Letter Reading) สร้าง Engagement สูงสุด
  • Multi-Camera Coverage
    การตัดสลับกล้องที่ลื่นไหลทำให้รู้สึกเหมือนนั่งอยู่ในสตูดิโอ ไม่ใช่แค่ดูจอ
  • Milestone Storytelling
    แฟนคลับต้องการ "เป็นพยาน" ให้กับช่วงเวลาสำคัญ — ยิ่งเป็นครั้งแรก ยิ่งต้อง Perfect
REFERENCE 02

What's YUPP! EP.29 — Trainee Debut Live

YUPP! Original

Live การเปิดตัว Trainee YUPP! ครั้งแรก — รูปแบบ Sit-Down Reveal ในสตูดิโอ YUPP! เป็น Blueprint สำหรับ MARQUISE โดยตรง

KEY LEARNINGS
  • Debut Live = ความทรงจำแรก
    คนดู 200+ เข้าร่วมตั้งแต่นาทีแรก — ทุกคนรู้ว่านี่คือ "ช่วงเวลาที่ย้อนกลับมาไม่ได้"
  • Studio Size ≠ Production Quality
    สตูดิโอเล็กของ YUPP! สร้าง Debut Live ที่ดูดีได้ด้วยการจัดกล้อง + แสงที่ถูกต้อง
  • Comment = Live Energy
    Host อ่าน Comment จากผู้ชมตลอดรายการ — ช่วยให้บรรยากาศในสตูดิโอไม่เงียบ
  • แฟนไทย + อินเตอร์ พร้อมเชียร์
    ฐานแฟน YUPP! พร้อมจะ "เกาะขอบจอ" ตั้งแต่ Live แรก — Moment นี้ต้องไม่พลาด
CORE INSIGHT FROM BOTH REFERENCES

แฟนคลับไม่ได้ดูแค่ "ศิลปิน" — พวกเขามาเพื่อ "เป็นส่วนหนึ่ง" ของช่วงเวลาประวัติศาสตร์

TECHNICAL QUALITY = RESPECT FOR THAT MOMENT
